# ラップタイム計測 Web アプリ

ブラウザ上で動くシンプルなストップウォッチ / ラップタイム計測アプリです。  
計測は `performance.now()` の実測差分を使うため、`setInterval` の遅延によるズレが出にくい実装になっています。

## デモ（動作イメージ）

- 合計時間（Total）
- 現在ラップ（Current Lap）
- ラップ記録一覧
- CSV 出力

## 使い方

1. `index.html` をブラウザで開きます
2. `スタート` で計測開始
3. `ラップ` でラップを記録（現在ラップが 0 に戻ります）
4. `ストップ` で停止
5. `リセット` で全クリア
6. `CSV出力` でラップ一覧を CSV ダウンロード

※ ローカルファイルとして開いても動きます（ビルド不要）。

## CSV 出力形式

ヘッダー: `Lap,Time`  
例:

```csv
Lap,Time
1,00:00:12.34
2,00:00:10.56
```

## ファイル構成

- `index.html` UI
- `style.css` スタイル
- `script.js` 計測ロジック / 操作 / CSV 出力

## 技術メモ（計測精度について）

一般に `setInterval(fn, 10)` は 10ms 間隔での呼び出しが保証されず、負荷やタブ非表示時に大きく遅延します。  
このアプリは「タイマーの回数 × 固定増分」ではなく、「開始時刻からの経過（実測）」で表示・確定する方式です。

## 開発

特別なセットアップは不要です。編集後はブラウザをリロードしてください。
