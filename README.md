# ラップタイム計測 Web アプリ
ブラウザ上で動くシンプルなストップウォッチ / ラップタイム計測アプリです。  

## デモ（動作イメージ）

- 合計時間（Total）
- 現在ラップ（Current Lap）
- ラップ記録一覧
- CSV 出力

## 使い方

1. `index.html` をブラウザで開きます
2. `スタート` で計測開始
3. `ラップ` でラップを記録（現在ラップが 0 に戻ります）
4. `ストップ` で停止
5. `リセット` で全クリア
6. `CSV出力` でラップ一覧を CSV ダウンロード

※ ローカルファイルとして開いても動きます（ビルド不要）。

## ホーム画面に追加して使う（スマホ）

通常のアプリのように使いたい場合は、以下の方法でホーム画面に追加できます。

- iPhone（Safari）
  1. Safari で `index.html` を開く
  2. 共有ボタン →「ホーム画面に追加」
  3. 追加後はホーム画面のアイコンから起動

- Android（Chrome）
  1. Chrome で `index.html` を開く
  2. 右上メニュー →「ホーム画面に追加」または「アプリをインストール」
  3. 追加後はホーム画面のアイコンから起動

## CSV 出力形式

ヘッダー: `Lap,Time`  
例:

```csv
Lap,Time
1,00:00:12.34
2,00:00:10.56
```
